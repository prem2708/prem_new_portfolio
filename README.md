## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/prem-mac.git
   cd prem-mac
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

   ```bash
   npm run dev
   ```
